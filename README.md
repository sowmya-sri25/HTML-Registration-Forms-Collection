This repository contains multiple HTML forms created using basic HTML elements.  
Each form demonstrates different input fields and form structures used in real-world web applications.

## 1. Registration Form
A basic user registration form used to collect personal information.

### Fields Included
- First Name
- Last Name
- Email
- Password
- Re-type Password
- Contact Number
- Gender Selection
- Submit Button

## 2. Gym Registration Form
A form designed for gym membership applications.

### Fields Included
- Name
- Gender Selection
- Email
- Phone Number
- Current Weight
- Desired Weight
- Height
- BMI
- Address
- City
- State
- Zip Code
- Country
- Personal Trainer Requirement (Yes/No)
- Gym Experience (Yes/No)
- Apply Button
